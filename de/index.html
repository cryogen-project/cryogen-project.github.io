Einfach

Ein Programm, um Seiten mit Clojure zu generiern, verpackt als Leiningen-Template. Es müssen keine Datenbanken generiert werden, es funktionert sofort.

Statisch

Klare Trennung von Inhalt und Layout erlaubt eine einfache Anpassung an Ihre Bedürfnisse. CSS und JavaScript können leicht hinzugefügt werden.

Automatisch

Sie müssen sich nicht darum k¨mmern, Ihre Posts zu verlinken oder Ihre sitemap und RSS Feed aktuell zu halten - Cryogen macht das für Sie.

Anfangen in Sekunden

  1. Installieren Sie Leiningen
  2. Starten Sie ein neues Projekt
  3. Starten Sie den server und rufen Sie localhost:3000 in Ihrem Browser auf
Schnellstart
~ $ lein new cryogen mein-blog
~ $ cd mein-blog
~/my-blog $ lein ring server
Started server on port 3000
